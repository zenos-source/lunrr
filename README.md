# Licensing API

## Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/verify?key=XXX&hwid=XXX` | Verify license key |
| POST | `/gen` | Generate new key |
| GET | `/keys` | List all keys |
| DELETE | `/key/:key` | Delete a key |

## Deploy on Railway

Click Deploy button or use `railway up`
